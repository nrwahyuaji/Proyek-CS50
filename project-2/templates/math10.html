Linear Simultaneous Equations with Two Variables

Linear Simultaneous Equations with Two Variables G-10

A finite set of linear equations in variables x and y is called a linear simultaneous (or a linear system). An ordered pair (x0, y0) is a solution of the system if (x0, y0) is a solution of every equation in the system. For example, (4, -2) is a solution to the following system: 3x + y = 10 5x - 2y = 24; Since (4, -2) satisfies both equations, (1,-7) is not a solution of the system since it satisfies only the first equation. The solution set of a system of linear equations is a set of ordered pairs that satisfies each of the linear equations. Solving The Linear Simultaneous Equations with Two Variables In general, a linear simultaneous equations with variables x and y can be expressed as follows. a1x + b1y = c1 a2x + b2y = c2 Where a1, b1, c1, a2, b2, and c2E R

There are several methods to solve a linear simultaneous equations:

  1. Graphing method
  2. Elimination method
  3. Substitution method
  4. Substitution-Elimination method
